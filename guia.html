Atributos

  • En HTML, los atributos son características que se usan para proporcionar información adicional sobre un elemento HTML.
    Se colocan dentro de la etiqueta de apertura del elemento y consisten en un nombre de atributo y un valor.
    Estos atributos controlan diversas propiedades del elemento, como su apariencia o su comportamiento.
  • Es posible asignar varios atributos a un mismo elemento, siempre separándolos por espacios dentro de la etiqueta.
    Por ejemplo, <a href="enlace" target="blank" class="enlace"> contiene tres atributos que controlan el destino del enlace, cómo se abre el enlace en una nueva pestaña y la clase CSS aplicada.
  • Hay varios atributos comunes en HTML.
    El atributo id permite identificar de manera única un elemento, lo que es útil para enlazar estilos CSS o scripts de JavaScript a un elemento específico.
    Por ejemplo: <div id="mi-div"> . El atributo class se utiliza para aplicar estilos a múltiples elementos mediante clases CSS.
    Un ejemplo sería <p class="texto-destacado"> , donde varios elementos pueden compartir la misma clase para recibir los mismos estilos.
  • En resumen, los atributos en HTML son esenciales para controlar y definir cómo se comportan o se ven los elementos dentro de una página web.
    Algunos atributos son comunes a todos los elementos, mientras que otros son específicos de ciertos tipos de elementos, lo que permite mayor flexibilidad en el diseño y la interacción de las páginas web.

Ejemplos

  1. href="" Se usa para redirigir a a un link especifico.
  2. src="" Se usa para proporcionar una fuente para una iamgen o vídeo.
  3. alt="" Se usa en las imagenes para proporcionar una descripcion cunado no carga la imagen.
  4. class="" Se utiliza para darle un nombre a una etiqueta y podes editarla desde CSS independientemente.
  5. id="" Se utiliza para lo mismo que la anterior pero es mas frecuente para usar con JavaScript.

Entidades

  • En HTML, las entidades son secuencias de caracteres especiales que representan símbolos que no se pueden escribir directamente en el código.
    Esto es útil para mostrar caracteres reservados o símbolos que podrían causar problemas en el navegador.
    Las entidades HTML siempre comienzan con un símbolo de ampersand (&) y terminan con un punto y coma (;).

Ejemplos

  1. &lt; < Se usa para sustituir el simbolo y representarlo en una web.
  2. &gt; > Se usa igual que su contraparte.
  3. &amp; & Se usa para hacer visible el simbolo que que acciona los atributos.
  4. &copy; © Se usa para poner el simbolo de copyright.
  5. &euro; € Se usa para poner el simbolo de euro.

Headers

  • Los Headers son etiquetas de texto que van de mayor tamaño a menor.

Ejemplos

  1. <h1> Se usa para Titulos.
  2. <h2> Se usa para Subtitulos.
  3. <h3> Se usa en textos resaltados.
  4. <h4> Se usa para texto normal.
  5. <h5> Se usa para cosas menos importantes.
  6. <h6> Es el mas pequeño de todos.

Listas

  • Las listas son una manera de organizar elementos para agruparlos en una lista con puntos o en una con numeros.
  • Las etiquetas de <ul> y <ol> solo organizan, la etiquetas de <li> es en la que se escribe el texto.

Ejemplos

  1. <ul> Las listas desordenadas o son listas que organizan con puntos o los atributos llamados &bullet;.
  2. <ol> Son lo mismo que las desordenadas pero usan numeros en vez de los puntos.
  3. <li> Es el elemento que se usa dentro de las listas para escribir el texto.

Etiquetas

  • Las etiquetas en HTML son la forma basica de escribir un codigo de manera que es la base para usar los atributos y demas formas de codigo.
  • Las etiquetas de <ul> y <li> son un ejemplo de etiquetas que se usan en conjunto.

Ejemplos

  1. <a> Su funcion es actuar como un hipervinculo el cual te redirige a otro sitio.
  2. <h1> Etiqueta usada mayormente para encabezados.
  3. <hr> Crea una linea que va de lado a lado de la pantalla.
  4. <br> Cambia de renglon, haciendo que en donde lo pongas se corte y siga en el siguiente renglon.
  5. <div> Es un contenedor que se usa para segregar contenido y modificarlo por separado.

Clases

  • Las clases <ejemplo class="ejemplo"> son un tipo de atributo que tiene el proposito de modificar al elemento desde CSS

Ejemplos

  1. .ejemplo { font-size: bold; } Este es un ejemplo de su sintaxis y funcion en el archivo de CSS.

Favicon

  • Los Favicon son iconos que se ponen en la pagina con la siguiente sintaxis, <link rel="shortcut icon" href="el favicon que descargaste" type="image/x-icon">.
  • El favicon se puede conseguir de muchas fuentes pero tiene que ser un formato de pocos pixeles y tienes que descargarlo para agregarlo a tu web.

Ejemplos

Iframes

  • Los <iframe> representan un navegador integrado en el que puedes mostrar variedad de codigos.
  • Los <iframe> se usan mayormenete para insertar videos de youtube en una pagina al hacer click en la opcion de embed en un video.

Ejemplos

Quotes

  • Las "Quotes" o <q> se usan para separar una pequeña porcion del codigo.
  • Las "Blockquotes" o <blockquote> se usan para separar porciones de codigo mas grandes.

Ejemplos

  1. <q> Es la que una acotacion corta.
  2. <blockquote> Es una acotacion mas larga usada para parrafos completos.

Formatos de texto

  • Los formatos de texto son modificadores incorporados en el html para evitar el uso de CSS en paginas simples.

Ejemplos

  1. <b> Cambia el texto a negrita
  2. <em> Cambia el texto a italica.
  3. <i> Cambia el texto a italica pero es con diferente sintaxis.
  4. <str> o <strong> Cambia el texto a negrita pero se usa en diferente situacion.
  5. <u> Le pone underline a el texto seleccionado.

Comentarios

  • Los comentarios son un tipo de codigo que no se muestra en la pagina, se usa para aclarar ciertos codigo o separar por secciones dentro del codigo.
  • Otro de sus usos es quitar las funciones a las porciones de codigo mientra se experimenta con otro.

Ejemplos

  1. <!----> En medio de los guiones se escribe el texto deseado.
  2. /**/ Esta es su variante para poner comentarios en CSS

File paths

  • Los file paths son rutas o direcciones que indican la ubicación exacta de un archivo dentro de un sistema de archivos.
  • Pueden ser absolutos (que empiezan desde la raíz del sistema) o relativos (basados en la ubicación actual del archivo).
  • Los path files ayudan a acceder, abrir o manipular archivos en el sistema, y pueden variar según el sistema operativo (Windows, Linux, macOS).
  • Normalmente incluyen directorios, subdirectorios y el nombre del archivo con su extensión.

Ejemplos

  1. href="/home/user/documents/report.pdf" Es una ruta absoluta para Linux
  2. href="C:\Users\John\Documents\report.pdf" Es una ruta absoluta para Windows
  3. href="./images/photo.jpg" Es una ruta relativa.
  4. href="../projects/code.js" Es una ruta relativa a un directorio.

Imágenes

  • La etiqueta <img> se usa para insertar imágenes en una página web.
  • Es importante proporcionar el atributo alt para describir la imagen, mejorando la accesibilidad.
  • Puedes ajustar el tamaño de la imagen usando los atributos width y height.

Ejemplos

  1. <img src="imagen.jpg" alt="Descripción de la imagen">
  2. <img src="logo.png" width="100" height="50" alt="Logo de la empresa">

HTML Semántico

  • El HTML semántico utiliza etiquetas que describen el propósito del contenido, mejorando la accesibilidad y el SEO.
  • Etiquetas como <header>, <footer>, <article> y <section> son ejemplos de HTML semántico.

Ejemplos

  1. <header> ... </header> Define el encabezado de una página o sección.
  2. <article> ... </article> Representa contenido independiente y autónomo.

Videos

  • La etiqueta <video> se usa para insertar videos en una página web.
  • Puedes controlar la reproducción del video usando atributos como controls, autoplay y loop.
  • Es recomendable proporcionar formatos alternativos para asegurar la compatibilidad entre navegadores.

Ejemplos

  1. <video src="video.mp4" controls></video>
  2. <video width="320" height="240" autoplay loop></video>

Elementos

  • Los elementos en HTML son las construcciones básicas que componen una página web, formados por etiquetas de apertura y cierre.
  • Cada elemento puede contener texto, otros elementos o ambos.
  • Ejemplos de elementos incluyen <div>, <p>, <a>, entre otros.

Ejemplos

  1. <div>Contenido</div> Define una sección genérica del documento.
  2. <p>Este es un párrafo.</p> Representa un párrafo de texto.

Formularios

  • La etiqueta <form> se utiliza para crear formularios que permiten a los usuarios enviar datos.
  • Dentro de un formulario, puedes usar elementos como <input>, <textarea>, <button>, entre otros.
  • Es importante definir el atributo action para indicar a dónde se enviarán los datos y method para el método de envío.

Ejemplos

  1. <form action="submit.php" method="post">
      <input type="text" name="nombre">
      <button type="submit">Enviar</button>
    </form>
  2. <form action="/buscar" method="get">
      <input type="search" name="q" placeholder="Buscar...">
      <button type="submit">Buscar</button>
    </form>

Links

  • La etiqueta <a> se usa para crear hipervínculos que conectan diferentes páginas o recursos.
  • El atributo href especifica la URL del destino del enlace.
  • Puedes abrir enlaces en una nueva pestaña usando el atributo target="_blank".

Ejemplos

  1. <a href="https://www.ejemplo.com">Visitar Ejemplo</a>
  2. <a href="https://www.ejemplo.com" target="_blank">Abrir en nueva pestaña</a>

Tablas

  • La etiqueta <table> se usa para crear tablas que organizan datos en filas y columnas.
  • Dentro de una tabla, se utilizan etiquetas como <tr> (fila), <th> (encabezado de columna) y <td> (celda de datos).
  • Es recomendable usar etiquetas semánticas para mejorar la accesibilidad y estructura de la tabla.

Ejemplos

  1. <table>
      <tr>
        <th>Nombre</th>
        <th>Edad</th>
      </tr>
      <tr>
        <td>Juan</td>
        <td>30</td>
      </tr>
    </table>
  2. <table border="1">
      <tr>
        <th>Producto</th>
        <th>Precio</th>
      </tr>
      <tr>
        <td>Laptop</td>
        <td>$1000</td>
      </tr>
    </table>
