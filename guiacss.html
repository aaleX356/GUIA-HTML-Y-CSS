Anatomía

  • La anatomía en CSS se refiere a la estructura básica de una regla CSS, que incluye el selector y las declaraciones.
  • Cada declaración dentro de una regla CSS consta de una propiedad y un valor, separados por dos puntos.
  • Las reglas CSS se encierran entre llaves para definir el bloque de estilos.

Ejemplos

  1. selector { propiedad: valor; } Ejemplo: p { color: blue; }
  2. h1 { font-size: 2em; }

Animaciones

  • Las animaciones en CSS permiten cambiar propiedades de un elemento de manera gradual.
  • Se definen usando las propiedades @keyframes y animation.
  • Las animaciones pueden ser infinitas o limitarse a un número específico de repeticiones.

Ejemplos

  1. @keyframes ejemploAnimacion {
      from { opacity: 0; }
      to { opacity: 1; }
    }
  2. .fade-in {
      animation: ejemploAnimacion 2s ease-in-out;
    }

Background

  • La propiedad background en CSS se usa para establecer el fondo de un elemento.
  • Puedes definir colores, imágenes, posiciones, y repeticiones para el fondo.
  • Existen propiedades específicas como background-color, background-image, background-position, y background-repeat.

Ejemplos

  1. background-color: #f0f0f0;
  2. background-image: url('fondo.jpg');
  3. background: #333 url('fondo.jpg') no-repeat center center;

Borders

  • Las propiedades de borde en CSS controlan los bordes de los elementos.
  • Puedes definir el grosor, el estilo y el color del borde.
  • Existen propiedades como border-width, border-style, y border-color.

Ejemplos

  1. border: 1px solid #000;
  2. border-radius: 5px; (para bordes redondeados)
  3. border-top: 2px dashed red;

Box Model

  • El Box Model es una caja que envuelve cada elemento HTML, consistiendo en márgenes, bordes, relleno y contenido.
  • Controla cómo los elementos se dimensionan y se posicionan en la página.
  • Comprender el Box Model es fundamental para diseñar layouts efectivos en CSS.

Ejemplos

  1. width: 200px; Define el ancho del contenido.
  2. padding: 10px; Agrega espacio interno alrededor del contenido.
  3. border: 1px solid #000; Añade un borde alrededor del elemento.
  4. margin: 20px; Añade espacio externo alrededor del elemento.

Box Sizing

  • La propiedad box-sizing controla cómo se calculan los anchos y altos de los elementos.
  • Los valores más comunes son content-box (por defecto) y border-box.
  • border-box incluye el padding y el borde dentro del ancho y alto especificados.

Ejemplos

  1. box-sizing: border-box;
  2. * {
      box-sizing: border-box;
    }     (Aplica box-sizing a todos los elementos)

Colors

  • La propiedad color define el color del texto de un elemento.
  • Otras propiedades relacionadas incluyen background-color y border-color.
  • Puedes usar diferentes formatos de color como nombres, hexadecimales, RGB, RGBA, HSL y HSLA.

Ejemplos

  1. color: red;
  2. color: #ff0000;
  3. color: rgb(255, 0, 0);
  4. color: rgba(255, 0, 0, 0.5);

Combinators

  • Los combinadores en CSS permiten seleccionar elementos basados en su relación con otros elementos.
  • Los combinadores más comunes son el combinador descendiente (espacio), hijo (>), adyacente (+) y general (~).
  • Facilitan la selección precisa de elementos para aplicar estilos específicos.

Ejemplos

  1. div p { color: blue; } Selecciona todos los p dentro de div.
  2. ul > li { list-style: none; } Selecciona solo los li que son hijos directos de ul.
  3. h1 + p { margin-top: 0; } Selecciona el p que está inmediatamente después de un h1.
  4. h1 ~ p { color: gray; } Selecciona todos los p que son hermanos de un h1.

Comments

  • Los comentarios en CSS se utilizan para agregar notas o descripciones dentro del código.
  • No afectan la presentación del documento y son ignorados por el navegador.
  • Se escriben entre /* y */.

Ejemplos

  1. /* Este es un comentario en CSS */
  2. /*
      Estilos para el encabezado
    */

Cursor

  • La propiedad cursor controla el tipo de cursor que se muestra cuando el usuario pasa el ratón sobre un elemento.
  • Existen varios valores como pointer, default, wait, text, entre otros.
  • Mejora la experiencia del usuario indicando interactividad.

Ejemplos

  1. cursor: pointer; (Cambia el cursor a una mano apuntando)
  2. cursor: wait; (Cambia el cursor a una rueda giratoria)
  3. cursor: text; (Cambia el cursor a una I para edición de texto)

Display

  • La propiedad display determina cómo se muestra un elemento en la página.
  • Los valores comunes incluyen block, inline, inline-block, flex, y none.
  • Controla el flujo y la disposición de los elementos en el layout.

Ejemplos

  1. display: block; (Elemento de bloque que ocupa toda la línea)
  2. display: inline; (Elemento en línea que no inicia una nueva línea)
  3. display: flex; (Activa el modelo Flexbox para el contenedor)
  4. display: none; (Oculta el elemento de la página)

Filter Functions

  • Las funciones de filtro en CSS permiten aplicar efectos visuales a elementos, como desenfoque, brillo, contraste, etc.
  • Se aplican usando la propiedad filter.
  • Mejoran la apariencia visual sin necesidad de editar las imágenes originales.

Ejemplos

  1. filter: blur(5px); (Aplica un desenfoque de 5 píxeles)
  2. filter: brightness(150%); (Aumenta el brillo al 150%)
  3. filter: grayscale(100%); (Convierte la imagen a escala de grises)
  4. filter: drop-shadow(16px 16px 20px gray); (Añade una sombra al elemento)

Flexbox

  • Flexbox es un modelo de layout en CSS que facilita la alineación y distribución de elementos dentro de un contenedor.
  • Se activa aplicando display: flex; al contenedor.
  • Permite una disposición flexible y adaptable, especialmente útil para diseños responsivos.

Ejemplos

  1. .contenedor {
      display: flex;
      justify-content: center;
      align-items: center;
    }
  2. .item {
      flex: 1;
    }

Floats

  • La propiedad float permite que un elemento flote a la izquierda o derecha dentro de su contenedor.
  • Utilizada comúnmente para diseños de múltiples columnas.
  • Requiere manejo de la limpieza (clear) para evitar problemas de layout.

Ejemplos

  1. float: left;
  2. float: right;
  3. clear: both; (Para limpiar ambos lados)

Grids

  • CSS Grid es un sistema de layout bidimensional que permite crear diseños complejos y responsivos.
  • Se activa aplicando display: grid; al contenedor.
  • Permite definir filas y columnas, y colocar elementos en áreas específicas del grid.

Ejemplos

  1. .grid-contenedor {
      display: grid;
      grid-template-columns: repeat(3, 1fr);
      grid-gap: 10px;
    }
  2. .grid-item {
      background-color: #f0f0f0;
      padding: 20px;
    }

Inheritance

  • La herencia en CSS permite que ciertas propiedades se transfieran de un elemento padre a sus elementos hijos.
  • No todas las propiedades son heredables; las que lo son incluyen color, font-family, y line-height.
  • Facilita la consistencia en estilos sin necesidad de repetir código.

Ejemplos

  1. body {
      color: #333;
      font-family: Arial, sans-serif;
    }     (Los elementos dentro del body heredarán estos estilos)
  2. .padre {
      font-size: 16px;
    }
    .hijo {
       /* Hereda font-size de .padre */
    }

Margins

  • La propiedad margin crea espacio alrededor de los elementos, fuera de sus bordes.
  • Puedes definir márgenes individuales para cada lado usando margin-top, margin-right, etc.
  • Los márgenes pueden colapsar en ciertos casos para evitar espacios dobles.

Ejemplos

  1. margin: 20px; (Aplica 20px de margen en todos los lados)
  2. margin-top: 10px;
  3. margin: 10px 20px; (10px arriba y abajo, 20px izquierda y derecha)

Math Functions

  • CSS proporciona funciones matemáticas como calc() para realizar cálculos dinámicos en propiedades.
  • Permite combinar unidades diferentes y realizar operaciones como suma, resta, multiplicación y división.
  • Mejora la flexibilidad en el diseño sin necesidad de preprocesadores.

Ejemplos

  1. width: calc(100% - 50px);
  2. height: calc(50vh + 20px);
  3. margin: calc(10px + 2em);

Overflow

  • La propiedad overflow controla qué sucede cuando el contenido de un elemento excede su tamaño.
  • Los valores comunes son visible, hidden, scroll, y auto.
  • Permite manejar el desbordamiento de contenido de manera eficiente.

Ejemplos

  1. overflow: hidden;
  2. overflow: scroll;
  3. overflow: auto; (Añade barras de desplazamiento si es necesario)

Padding

  • La propiedad padding crea espacio interno dentro de un elemento, entre el contenido y el borde.
  • Puedes definir padding individual para cada lado usando padding-top, padding-right, etc.
  • El padding afecta el tamaño total del elemento a menos que se use box-sizing: border-box;.

Ejemplos

  1. padding: 15px; (Aplica 15px de padding en todos los lados)
  2. padding-top: 10px;
  3. padding: 10px 20px; (10px arriba y abajo, 20px izquierda y derecha)

Position

  • La propiedad position controla cómo se posiciona un elemento en la página.
  • Los valores incluyen static, relative, absolute, fixed, y sticky.
  • Permite colocar elementos en ubicaciones específicas dentro del layout.

Ejemplos

  1. position: relative; (Posiciona el elemento relativo a su posición original)
  2. position: absolute; (Posiciona el elemento relativo a su ancestro posicionado más cercano)
  3. position: fixed; (Posiciona el elemento relativo a la ventana del navegador)
  4. position: sticky; (Mantiene el elemento pegado al viewport mientras se desplaza)

Pseudo-classes

  • Las pseudo-clases en CSS permiten seleccionar elementos en estados específicos o basados en su posición.
  • Ejemplos comunes incluyen :hover, :active, :nth-child(), y :first-of-type.
  • Facilitan la aplicación de estilos dinámicos sin necesidad de JavaScript.

Ejemplos

  1. a:hover { color: red; } (Cambia el color del enlace al pasar el ratón)
  2. li:nth-child(2) { background-color: #f0f0f0; } (Aplica fondo a cada segundo li)
  3. button:active { transform: scale(0.95); } (Escala el botón al ser clicado)

SASS

  • SASS (Syntactically Awesome Stylesheets) es un preprocesador de CSS que añade funcionalidades avanzadas como variables, anidamiento, mixins, y más.
  • Permite escribir CSS de manera más eficiente y mantenible.
  • Se compila en CSS estándar que los navegadores pueden interpretar.

Ejemplos

  1. $color-principal: #3498db;
    .boton {
      background-color: $color-principal;
    }
  2. @mixin flex-center {
      display: flex;
      justify-content: center;
      align-items: center;
    }
    .contenedor {
      @include flex-center;
    }

Selectors

  • Los selectores en CSS especifican qué elementos HTML serán estilizados.
  • Tipos de selectores incluyen selectores de tipo, clase, ID, atributos, descendientes, combinadores, y más.
  • Permiten aplicar estilos precisos a elementos específicos en la página.

Ejemplos

  1. p { color: blue; } (Selecciona todos los párrafos)
  2. .activo { background-color: yellow; } (Selecciona elementos con la clase 'activo')
  3. #header { height: 100px; } (Selecciona el elemento con ID 'header')
  4. input[type="text"] { border: 1px solid #ccc; } (Selecciona inputs de tipo texto)

Sizing

  • Las propiedades de sizing en CSS controlan el tamaño de los elementos.
  • Incluyen width, height, max-width, max-height, min-width, y min-height.
  • Permiten definir dimensiones precisas o relativas para elementos.

Ejemplos

  1. width: 100px;
  2. height: 50vh;
  3. max-width: 500px;
  4. min-height: 200px;

Transform Functions

  • Las funciones de transformación en CSS permiten rotar, escalar, inclinar y trasladar elementos.
  • Se aplican usando la propiedad transform.
  • Mejoran la interactividad y animaciones visuales sin alterar el flujo del documento.

Ejemplos

  1. transform: rotate(45deg);
  2. transform: scale(1.5);
  3. transform: translateX(50px);
  4. transform: skewY(10deg);

Transition

  • Las transiciones en CSS permiten cambiar propiedades de manera gradual en respuesta a interacciones.
  • Se definen usando las propiedades transition-property, transition-duration, transition-timing-function, y transition-delay.
  • Mejoran la experiencia del usuario añadiendo fluidez a los cambios de estilo.

Ejemplos

  1. transition: background-color 0.5s ease;
  2. .boton:hover {
      background-color: green;
      transition: background-color 0.3s ease-in-out;
    }
  3. transition: all 1s;

Typography

  • La tipografía en CSS se refiere al estilo y presentación del texto.
  • Propiedades clave incluyen font-family, font-size, font-weight, line-height, y text-align.
  • Permite mejorar la legibilidad y estética del contenido textual.

Ejemplos

  1. font-family: 'Arial, sans-serif';
  2. font-size: 16px;
  3. font-weight: bold;
  4. line-height: 1.5;
  5. text-align: center;

Units

  • Las unidades en CSS determinan las dimensiones de las propiedades como tamaño, margen, padding, etc.
  • Las unidades más comunes incluyen px, em, rem, %, y vh/vw.
  • Elegir la unidad adecuada es crucial para diseños responsivos y accesibles.

Ejemplos

  1. width: 100px;
  2. margin: 2em;
  3. font-size: 1.5rem;
  4. height: 50vh;
  5. padding: 10%;

Variables

  • Las variables en CSS, conocidas como Custom Properties, permiten almacenar valores que pueden reutilizarse en el código.
  • Se definen dentro de un selector usando --nombre-variable y se acceden con la función var().
  • Facilitan la gestión y mantenimiento de estilos, especialmente en proyectos grandes.

Ejemplos

  1. :root {
      --color-principal: #3498db;
    }
  2. .boton {
      background-color: var(--color-principal);
    }

Visibility

  • La propiedad visibility controla si un elemento es visible o invisible en la página.
  • Los valores principales son visible, hidden, y collapse.
  • A diferencia de display: none;, visibility: hidden; mantiene el espacio del elemento en el layout.

Ejemplos

  1. visibility: hidden;
  2. visibility: visible;
  3. visibility: collapse; (Usado principalmente en tablas)
